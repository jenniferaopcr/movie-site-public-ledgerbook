# movie-site-public-ledgerbook
本仓库用于对互联网上已公开存在的影视类网站进行账册式整理，重点围绕电影与电视剧相关站点的基础公开信息进行系统汇总。通过统一规范记录站点名称、访问地址、站点定位说明及基础分类信息，便于长期保存与集中查阅。本项目不涉及任何影视资源的存储、播放、下载或解析，仅针对公开网络环境中可获取的信息进行整理与留存。所有内容均来源于公开渠道，整理过程保持客观、中立，仅作为资料整理与信息参考用途存在。
